# 📢 Sistem Pengaduan Masyarakat

Platform pengaduan masyarakat berbasis web menggunakan **PHP Native + MySQL + Tailwind CSS** dengan fitur modern seperti:

* 🔐 Multi Role Login (Admin & User)
* 🌙 Dark / Light Mode
* 📊 Dashboard Admin Modern
* 📈 Statistik & Chart Pengaduan
* 💬 Chat / Komentar Admin & User
* 📍 Tracking Pengaduan
* 📷 Upload Bukti Pengaduan
* ⚡ UI Modern Glassmorphism

---

# ✨ Tampilan Aplikasi

## 🏠 Home

Modern landing page dengan statistik dinamis.

## 👤 User

* Buat pengaduan

* Chat dengan admin
* Upload bukti

## 🛠 Admin

* Dashboard modern

# ⚙️ Cara Install

## 1️⃣ Clone Project

```bash
git clone https://github.com/username/pengaduan-app.git
```

---

## 2️⃣ Pindahkan ke htdocs

```bash
xampp/htdocs/pengaduan-app
```

---

## 3️⃣ Import Database

Import file SQL ke phpMyAdmin.

Database:

```bash
pengaduan_db
```

---

## 4️⃣ Konfigurasi Database

Edit file:

```bash
config/koneksi.php
```

---

## 5️⃣ Jalankan

```bash
http://localhost/pengaduan-app
```
